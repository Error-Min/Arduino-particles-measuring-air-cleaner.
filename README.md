* 12V 입력 커넥터를 사용하여 제작된 미세먼지 측정 공기청정기 입니다.   
* * 자동버튼을 누를시 미세먼지를 측정하여 미세먼지의 수치에 따라 네오픽셀 LED색이 변경됩니다.   
* * 자동버튼을 두 번누를시 네오픽셀이 꺼지며 미세먼지의 수치에 따라 쿨팬의 속도가 달라집니다.  
*  * 수동버튼은 1단계부터 3단계로 구성되어 있습니다.
